Sensitivity kernel（感度カーネル）作成の準備

今回はshbp, shfpを用いて計算してもらった偏微分係数波形を用いてtransverse成分のS及びScSの感度カーネルをプロットします.

用意するもの：BP, FPの偏微分係数波形、timewindow(時間窓)の情報、プロット用スクリプト

  1. timewindowの情報の作成
    • 作業場所はsensitivityの直下. 201506231218A/には自分が計算した震央距離の理論波形が入っていることが必須.
    • kibrary_propertyでTimewindowMaker(18)を選択
    • timewindowmaker.propertiesの中身を以下のように編集
     manhattan TimewindowMaker

     ##Path of a working folder (.)

     #workPath

     ##SacComponents to be used (Z R T)

     components T

     ##TauPPhases exPhases (sS)

     exPhases sS sScS sSdiff

     ##TauPPhases usePhases (S)

     usePhases S ScS Sdiff

     ##time before first phase. If it is 10, then 10 s before arrival (0)

     frontShift

     ##double time after last phase. If it is 60, then 60 s after arrival (0)

     rearShift 10
    • kibrary_operation timewindowmaker.propertiesでTimewindowMakerを実行
    • 作成されたtimewindow.datの中身の確認は以下のコマンドで. java io.github.kensuke1984.kibrary.timewindow.TimewindowInformationFile timewindow.dat
  2. 偏微分係数データの作成(Cross-correlation、FFT等)
    • sensitivity/threedPartialで作業を行う
    • kibrary_propertyでPartialDatasetaMaker(11)を選択
    • PartialDatasetMaker.propertiesを編集
    manhattan PartialDatasetMaker

    ##Path of a working folder (.)

    #workPath

    ##SacComponents to be used (Z R T)

    components T

    ##Path of a back propagate spc folder (BPinfo)

    #bpPath

    ##Path of a forward propagate spc folder (FPinfo)

    #fpPath

    ##String if 'modelName' is PREM, spectrum files in 'eventDir/PREM'   are used.

    ##If it is unset, then automatically set as the name of the folder in the eventDirs

    ##but the eventDirs can have only one folder inside and they must be same.

    #modelName

    ##Type source time function 0:none, 1:boxcar, 2:triangle. (0)

    ##or folder name containing *.stf if you want to your own GLOBALCMTID.stf 

    sourceTimeFunction 0

    ##Path of a time window file, must be set

    timewindowPath timewindow.dat

    ##PartialType[] compute types (MU)

    partialTypes MU

    ##double time length DSM parameter TLEN, must be set

    #TLEN 3276.8

    ##int step of frequency domain DSM parameter NP, must be set

    #NP 512

    ##double minimum value of passband (0.005)

    minFreq 0.05

    ##double maximum value of passband (0.08)

    maxFreq 0.1

    #double (20)

    #partialSamplingHz cant change now

    ##double SamplingHz in write dataset (1)

    #finalSamplingHz

    ##perturbationPath, must be set

    #perturbationPath perturbationPoint.inf

    ##File for Qstructure (if no file, then PREM)

    #qinf
    • kibrary_operation partialdatasetmaker.propertiesで実行
